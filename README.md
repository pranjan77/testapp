# docker-tomcat-tutorial
A basic tutorial on running a web app on Tomcat using Docker

# Steps
* Install [Docker](https://docs.docker.com/install/).
* Clone this repository 
* cd 'testapp'
* docker-compose up --build
* http://localhost:5000
